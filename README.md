# Engineering Projects Portfolio

This repository contains selected academic engineering projects completed during my Electrical and Electronics Engineering studies at Abdullah Gül University.

The projects cover electronics, signal processing, embedded systems, experimental analysis, robotics, and artificial intelligence. Each project folder includes technical reports, source code, simulation files, experimental results, and supporting documentation where available.

## Featured Project

### Enhancing Robot Navigation with Vision-Language Models

An ongoing senior design project focused on improving autonomous indoor robot navigation through Vision-Language Models, semantic mapping, and natural language-guided navigation.

The current implementation uses Habitat-Sim and VLMaps to generate open-vocabulary semantic maps from RGB-D observations and support language-based target localization.

**Main Topics**

- Vision-Language Models
- Computer Vision
- Semantic Mapping
- Autonomous Navigation
- Habitat-Sim
- VLMaps
- Python
