# Enhancing Robot Navigation with Vision-Language Models

## Project Overview

This ongoing senior design project investigates how Vision-Language Models (VLMs) can improve autonomous indoor robot navigation through semantic scene understanding and natural language-guided navigation.

The project integrates Habitat-Sim and VLMaps to generate open-vocabulary semantic maps from RGB-D observations, enabling robots to identify objects and navigate using natural language commands.

---

## Objectives

- Develop semantic maps from RGB-D observations
- Enable natural language-guided navigation
- Investigate open-vocabulary object localization
- Evaluate navigation performance in simulated environments
- Prepare the framework for future deployment on a real mobile robot
