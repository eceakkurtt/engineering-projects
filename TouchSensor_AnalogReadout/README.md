# Touch Sensor Interface & Analog Circuit Design

## Project Overview

This project presents the design and implementation of an analogue capacitive sensing interface capable of converting very small capacitance variations into measurable DC voltages.

The system was designed, simulated, and experimentally validated using transistor-level analogue circuits.

---

## Main Contributions

- Designed transistor-level analogue circuits
- Developed capacitive sensing interface
- Performed LTspice simulations
- Validated simulation results experimentally
- Investigated circuit sensitivity and stability
