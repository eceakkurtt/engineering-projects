# Digital Stethoscope System

## Project Overview

This project focused on the design and implementation of a digital stethoscope system for biomedical signal acquisition, visualization, and frequency analysis.

The system combines analog signal conditioning, Arduino-based data acquisition, and MATLAB-based signal processing to visualize heart sounds and analyze their frequency content using Fast Fourier Transform (FFT).

---

## Main Contributions

- Designed analog filter circuits for heart sound acquisition
- Implemented Arduino-based data acquisition
- Developed a MATLAB graphical user interface (GUI)
- Performed FFT-based frequency analysis
- Evaluated the system through experimental measurements
