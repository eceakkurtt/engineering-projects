# Helmholtz Coil System

## Project Overview

This project focused on the design and experimental evaluation of a Helmholtz coil system capable of producing a nearly uniform magnetic field.

Simulation results were compared with laboratory measurements.

---

## Main Contributions

- Designed Helmholtz coil system
- Simulated magnetic field generation
- Performed laboratory measurements
- Compared theoretical and experimental results
- Validated simulation accuracy

---

## Technologies
